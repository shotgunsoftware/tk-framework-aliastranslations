The Alias Translations FrameworkÂ¶

The Alias Translations Framework contains a collection of helpers and tools to make it easy and painless to convert Alias files to different formats.

Warning

This framework is a wrapper of Alias command line tools. This requires that you have Alias installed on the computer executing this code.

Note

For now, you can only convert files to these formats:
  • .wref
  • .jt
  • .CATPart
  • .igs
  • .stp
